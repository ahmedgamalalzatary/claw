# Claw Gateway

MVP scaffold for a personal WhatsApp AI gateway.

## Quick Start

1. Install dependencies:
   - `npm install`
2. Edit `config.json` with your Google API key and model names.
3. Run in dev:
   - `npm run dev`
4. Build and run prod:
   - `npm run build`
   - `npm start`

## Current Status

- Layered folder structure is ready.
- Infra files are ready (`Dockerfile`, `docker-compose.yml`, `config.json`).
- Core modules are placeholders for incremental implementation.
